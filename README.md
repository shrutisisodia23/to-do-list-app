# To Do App

A simple and clean Todo List application built using React and Vite, with features like adding tasks, deleting tasks, viewing task details, and dark/light mode switching. All data is stored in localStorage, so your tasks stay saved even after refreshing the page.

# Features 
Add new todo items, 
Delete todos,
View full details of a todo,
Dark/Light mode toggle,
Todos saved in browser localStorage,
Clean component structure

# Tech Stack
React + Vite, JavaScript (ES6+), Bootstrap, localStorage for data persistence

# User guide
1. Open the app in your browser.
2. Click "New+" to create a new task.
3. Tasks will automatically appear on the homepage.
4. Click a todo to view full details.
5. Toggle Dark/Light mode from the navbar.
6. Your todos will remain saved because of localStorage.
